# B2B Warehouse & POS API 

Микросервис для управления товарной номенклатурой, складскими запасами и кассовыми операциями (Point of Sale). Разработан как изолированный бэкенд-компонент для интеграции в ERP/CRM системы малого и среднего бизнеса.

## Ключевой функционал
* **Управление каталогом:** CRUD операции для товаров и категорий, поддержка штрихкодов.
* **Финансовая логика:** Безопасная работа с валютами и ценообразованием (наценки, скидки, расчет финальной стоимости) с использованием фиксированной точности `Decimal`.
* **Складские операции:** Фиксация приходов (Receipts) и списаний (Write-offs) с привязкой к поставщикам.
* **Медиа-хранилище:** Интеграция с AWS S3 для надежного хранения изображений товаров (с Graceful Degradation при недоступности облака).

## Технологический стек
* **Язык/Фреймворк:** Python 3.12, FastAPI
* **База данных:** PostgreSQL, SQLAlchemy 2.0 (async), Alembic (миграции)
* **Инфраструктура:** Docker, Docker Compose
* **Сторонние сервисы:** AWS S3 (boto3)

## Быстрый старт (Запуск через Docker)

1. Клонируйте репозиторий:
```bash
git clone <ссылка_на_твой_будущий_репозиторий>
cd <название_папки>
```

2. Создайте файл переменных окружения на основе примера:
```bash
cp .env.example .env
```

3. Запустите контейнеры:
```bash
docker-compose up --build
```

Сервис будет доступен по адресу: `http://localhost:8000`
Интерактивная документация API (Swagger): `http://localhost:8000/docs`
